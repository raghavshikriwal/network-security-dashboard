# 🛡️ Network Security Dashboard

A real-time cybersecurity monitoring platform built for Security Operations Centers (SOC).

## 🚀 Live Demo
[View Live Dashboard](https://network-sentinel-dashboard--raghavshikriwal.replit.app)

## 📋 Features
- 🔐 JWT Authentication System
- 📊 Real-time Network Traffic Charts
- 🌍 Live World Threat Map
- 🚨 Automated Alert & Notification System
- 🔍 IP Lookup & Analysis
- 📝 Incident Log Management
- ⚡ Socket.io Live Updates

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Real-time:** Socket.io
- **Database:** PostgreSQL
- **Auth:** JWT
